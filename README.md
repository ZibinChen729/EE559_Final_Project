# Heart Disease Prediction

This project predicts whether a patient has heart disease using three supervised learning models:

- Logistic Regression
- Decision Tree
- Multi-Layer Perceptron

## Files

This repository includes:

- `HeartDiseaseTrain-Test.csv`: dataset file
- `EE559_project.py`: Python script with the full code
- `README.md`: instructions for running the project

## How to Run
- Download or clone this repository.
- Make sure `HeartDiseaseTrain-Test.csv` is in the same folder as `EE559_project.py`.
- Run `EE559_project.py` using Python.
