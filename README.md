# Heart Disease Prediction

This project predicts whether a patient has heart disease using three supervised learning models:

- Logistic Regression
- Decision Tree
- Multi-Layer Perceptron

## Files

This repository includes:

- `HeartDiseaseTrain-Test.csv`: dataset file
- `heart_disease_prediction.ipynb`: Jupyter Notebook with the full code
- `README.md`: instructions for running the project

## How to Run
- Download or clone this repository.
- Make sure HeartDiseaseTrain-Test.csv is in the same folder as heart_disease_prediction.ipynb.
- Open heart_disease_prediction.ipynb using Jupyter Notebook, JupyterLab, or Google Colab.
- Run all cells from top to bottom.
