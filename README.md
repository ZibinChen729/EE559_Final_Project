# Heart Disease Prediction

This project predicts whether a patient has heart disease using three supervised learning models:

- Logistic Regression
- Decision Tree
- Multi-Layer Perceptron

## Files

This repository includes:

- `HeartDiseaseTrain-Test.csv`: dataset file
- `heart_disease_prediction.ipynb`: Jupyter Notebook with the full code
- `README.md`: instructions for running the project

## How to Run
1: Download or clone this repository.
2: Make sure HeartDiseaseTrain-Test.csv is in the same folder as heart_disease_prediction.ipynb.
3: Open heart_disease_prediction.ipynb using Jupyter Notebook, JupyterLab, or Google Colab.
4: Run all cells from top to bottom.
